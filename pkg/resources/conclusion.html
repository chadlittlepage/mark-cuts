Installation Complete

Two manual steps remain inside DaVinci Resolve:

  1. Enable external scripting
    Open Resolve, go to Preferences > System > General, set External scripting using to Local, click Save, then restart Resolve.
  2. Run the script
    Open a project with a timeline, then go to Workspace > Scripts > Edit > mark_cuts.

If the script doesn't appear

  • Restart Resolve after install.
  • Confirm Python 3 is installed from python.org (not Homebrew).
  • Confirm external scripting is set to Local.
