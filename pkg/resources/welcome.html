mark_cuts v1.0.1

DaVinci Resolve cut-point marker script

Adds a blue marker at every cut point on your current timeline, across all video tracks. Re-runnable and non-destructive.

What it does:

  • Walks every video track on the active timeline
  • Drops a blue Cut marker at each clip in / out point
  • Skips frames that already have a marker

This installer will:

  • Install the script for every Resolve page (Edit, Color, Fusion, Deliver)
  • Mirror it to every user account on this Mac
  • Check for Python 3 from python.org
  • Drop a Mark_Cuts_NEXT_STEPS.txt on your Desktop with the two manual Resolve steps

Requires: DaVinci Resolve 17+, Python 3 from python.org, macOS 12+.
